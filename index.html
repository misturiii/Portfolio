Link-Up

During the 36-hour hackathon SheHacks +7, we created a web application Link-Up. This website takes in your group's personal calendars and shows you all the available time slots. The user can then pick a time and date for their event. The plan will be downloaded into a calendar file that can be added to everyone's calendar. The website is built with HTML, CSS, and JavaScript.

MIMISushi is a responsive, mobile-friendly online sushi-ordering website utilizing JavaScript, HTML, CSS, and Vite.
